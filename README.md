# Свадебное приглашение — Алексей & София

Одностраничный адаптивный сайт-приглашение на Next.js/vinext. Внутри: первый экран, история пары, программа дня, локация, дресс-код и RSVP-форма.

## Быстрый старт

Понадобится Node.js 22.13 или новее.

```bash
npm install
npm run dev
```

Локальная версия откроется по адресу `http://localhost:3000`.

## Что заменить перед публикацией

Главные данные собраны в объекте `wedding` в начале `app/page.tsx`:

- имена пары;
- дата;
- название и адрес площадки;
- ссылка на карту.

Там же ниже находятся программа дня и все тексты приглашения. Цвета и шрифты задаются переменными в начале `app/globals.css`.

RSVP-форма сейчас работает как интерактивный демо-ответ в браузере. Чтобы ответы гостей сохранялись, подключите Formspree, Netlify Forms, Google Forms или собственный обработчик перед публикацией.

## Проверка и публикация

```bash
npm run build
```

Проект можно хранить в личном GitHub/GitLab-репозитории и развернуть через Cloudflare Pages, Vercel или другой сервис с поддержкой Next.js. Build command: `npm run build`.

```bash
git init
git add .
git commit -m "Wedding invitation"
git branch -M main
git remote add origin <адрес-вашего-репозитория>
git push -u origin main
```
