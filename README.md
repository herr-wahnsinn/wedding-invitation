# Свадебное приглашение — Андрей & Лиля

Одностраничный адаптивный сайт-приглашение на Next.js. Внутри: первый экран, история пары, программа дня, локация, дресс-код и RSVP-форма.

## Быстрый старт

Понадобится Node.js 22.13 или новее.

```bash
npm install
npm run dev
```

Локальная версия откроется по адресу `http://localhost:3000`.

## Что заменить перед публикацией

Главные данные собраны в объекте `wedding` в начале `app/page.tsx`:

- имена пары;
- дата;
- адреса ЗАГСа и ресторана;
- ссылки на карты.

Там же ниже находятся программа дня и все тексты приглашения. Цвета и шрифты задаются переменными в начале `app/globals.css`.

RSVP-форма сохраняет ответы в Google Sheets через приватный Google Apps Script. Персональный код берётся из параметра `?invite=...`, после чего сайт подставляет обращение к одному гостю или группе и выбирает форму «ты»/«вы». Серверная часть использует две переменные окружения из `.env.example`.

Приватные файлы настройки, список первых гостей и инструкция находятся в папке `outputs`; она исключена из Git. Не публикуйте персональные ссылки и общий секрет в открытом репозитории.

## Проверка и публикация

```bash
npm run build
```

Проект можно хранить в личном GitHub/GitLab-репозитории и развернуть через Vercel или другой сервис с поддержкой Next.js. В Vercel достаточно импортировать репозиторий: фреймворк и команда `npm run build` определятся автоматически.

```bash
git init
git add .
git commit -m "Wedding invitation"
git branch -M main
git remote add origin <адрес-вашего-репозитория>
git push -u origin main
```
